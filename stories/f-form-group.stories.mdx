# f-form-group

<br />
<br />

## Overview

<hr />

Use f-form-group to create logical sections within the form.

f-form-group is used to group multiple instances of an input element.

## variant

<hr />

Variant decides whether the input elements in a group have some gap or not when they are aligned horizontally.

<p class="note-section">
	Note: If variant is normal then the gap between elements is controlled by the parent group.
</p>
