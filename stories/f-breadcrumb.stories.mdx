### crumbs

<f-divider></f-divider>

Crumbs property represents the array of crumbs in a breadcrumb.

<br />

<Canvas>
	<Story name="crumbs">
		{args => {
			const data = [
				[
					{ tabIndex: 0, title: "Label 1 New Label Demo test" },
					{ tabIndex: 1, title: "Label 2" },
					{ tabIndex: 2, title: "Label 3" },
					{ tabIndex: 3, title: "Label 4" }
				],
				[
					{ tabIndex: 0, title: "Label 1" },
					{ tabIndex: 1, title: "Label 2" },
					{ tabIndex: 2, title: "Label 3" },
					{ tabIndex: 3, title: "Label 4" },
					{ tabIndex: 4, title: "Label 5" }
				]
			];
			return html`<f-div padding="small" gap="x-large">
				${data.map(
					item =>
						html` <f-div direction="column" gap="large"
							><f-text>${item.length > 4 ? "Crumbs length > 4" : "Crumbs length < 4"}</f-text
							><f-breadcrumb .crumbs=${item}></f-breadcrumb
						></f-div>`
				)}
			</f-div>`;
		}}
