## state

<hr />

States on f-div are used to communicate purpose and its connotation. For example, a red color
connotes danger, whereas a green color connotes success and so on.

<p class="note-section">
	Note: In case of nested f-div, all children f-div inherit the state from parent f-div until
	specified.
</p>

## overflow

<hr />

Overflow property defines the behavior of the overflowing elements inside a f-div. It can happen
either if layout is overflowing from the parent container or layout has fixed width/height and the
elements are overflowing.

<br />

Elements can behave in 3 ways in this situation:

- Wrap: Overflowing elements are wrapped to the next row/column. This is the default behavior.
- Scroll: Layout becomes a scrollable component and the overflowing elements are accessible on scroll.
- Hidden: Overflowing elements are clipped and remain hidden.
