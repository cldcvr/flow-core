# f-button

<br />

## Overview

<f-divider></f-divider>

### Do

- To enable a user action

### Don't

- To navigate to another page, use text as link
- If you need an icon only button use the Icon button
- To show a hybrid of an action and a dropdown, use Split button

## counter

<f-divider></f-divider>

The counter component by default abbreviates the value (1000 = 1k). You can learn more about the counter
[here](/docs/components-f-counter--anatomy).
